# RocketTerminal

RocketTerminal replaces the old desktop debugging GUI.

Features:

- Rich terminal output.
- Typer CLI.
- QR generation.
- Authenticated websocket server.
- Connection status.
- Received task display.
- Latency display.
- Nemotron status.
- API health summary.

Run:

```powershell
python -m agent.main terminal --host 0.0.0.0 --port 8765
```
