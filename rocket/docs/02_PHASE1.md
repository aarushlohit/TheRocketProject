# Phase 1 Scope

Phase 1 is complete when:

- Voice audio reaches Nemotron and returns one task.
- Drawing PNG reaches Nemotron and returns one task.
- Braille text reaches Nemotron and returns one task.
- RocketTerminal displays each generated task with source and latency.

Out of scope:

- Executing the task.
- Confirming dangerous actions.
- Automating Windows, browsers, files, or applications.
- Keeping memories or long-running plans.
- Modifying OpenCode internals.
