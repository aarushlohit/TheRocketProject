# Rocket V3 Vision

Rocket Phase 1 is a blind-first input bridge.

The phone captures voice, drawing, or Braille input. Nemotron Omni converts that perception into one executable task string. RocketTerminal displays the task on the laptop.

Phase 1 stops there. It does not automate the laptop, execute commands, control windows, browse, use MCP, store memory, or run verification loops.
