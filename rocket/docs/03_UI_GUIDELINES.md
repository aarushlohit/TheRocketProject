# UI Guidelines

Rocket mobile is blind-first.

- Single tap speaks what the control is.
- Double tap continues or performs the action.
- Primary controls have at least 120 dp height.
- Text is high contrast.
- TTS is used for connection, scanning, recording, processing, task generation, send success, errors, reconnects, and lost connection.
- Haptics support touch confirmation but do not replace speech.

Home screen cards:

- Voice
- Drawing
- Braille
- Settings
