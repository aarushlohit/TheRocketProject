# Requirements

- Python 3.11+
- Flutter SDK
- Android Developer Mode on Windows for Flutter plugin symlinks
- `NVIDIA_API_KEY` for Nemotron
- Network access to `https://integrate.api.nvidia.com/v1`
- Network access to Pollinations fallback

Install Python dependencies:

```powershell
python -m pip install -r requirements.txt
```

Install Flutter dependencies:

```powershell
cd mobile_app
flutter pub get
```
